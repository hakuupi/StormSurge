# StormSurge
Repository for sharing code between Emily, Hannah, Logan and Dr. Mayo for our 2022 Emory REU project.
These files contain all the code and data used in our project, along with our [website](https://github.com/loganpknudsen/emory-reu-ret-website/blob/main/content/projects/2022-storm-surge/index.md) discussing our research. [Here](https://www.math.emory.edu/site/cmds-reuret/summer2022/) is the link to the Summer 2022 Emory REU/RET Computational Mathematics for Data Science of which this project was a part of.
